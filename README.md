# Mohamed Wahed — Portfolio (2026)

A modern, minimalist, fully responsive personal portfolio with a soft gradient theme, smooth scrolling, subtle animations, and a dark mode toggle.

## Tech

- HTML
- Tailwind CSS (CDN)
- JavaScript (vanilla)

## Run

Because this is a static site, you can:

- Open `index.html` directly in your browser, or
- Use any local server (recommended so links/assets behave consistently).

Example (if you have Python):

```bash
python -m http.server 5173
```

Then visit `http://localhost:5173`.

## Customize

- **Text/content**: edit `index.html`
- **Theme + components styling**: `src/styles.css`
- **Interactions** (dark mode, mobile menu, animations): `src/main.js`
- **Project links**: replace the `href="#"` in the Projects + Social buttons with your real URLs.

## Notes

- The contact form is a demo (does not send emails). You can connect it to EmailJS, Formspree, or your own backend when ready.
